# Nearshow-FSAD-project

Overview
NearShow is a full-stack web application with a separate frontend (client) and backend (server).
It is structured to maintain scalability and clean architecture.

Features
Full-stack architecture
Separate client and server
Organized folder structure
API-based communication

Installation
Clone the repository
   git clone https://github.com/Nishanthreddy69/Nearshow-FSAD-project
Install dependencies
   For client:
     cd client
     npm install
   For server:
     cd ../server
     npm install

Running the Project
Start backend:
  cd server
  npm start
Start frontend:
  cd client
  npm start
  
Technologies Used
Frontend: React.js
Backend: Node.js, Express.js
